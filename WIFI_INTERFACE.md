# 📡 WiFi Web Interface для Solar Tracker

## 🎯 Возможности

Веб-интерфейс позволяет управлять солнечным трекером через браузер:

### 📊 Отображаемые показания
- **Left Panel Voltage** - напряжение левой солнечной панели (мВ)
- **Right Panel Voltage** - напряжение правой панели (мВ)
- **Difference Meter** - визуализация разницы напряжений между панелями
- **Motor Status** - текущее состояние мотора (STOP, LEFT ↙, RIGHT ↗)
- **Mode Indicator** - текущий режим работы (AUTO, MANUAL, CALIBRATION)

### 🎮 Кнопки управления
1. **← LEFT** - поворот панелей влево на один импульс
2. **RIGHT →** - поворот панелей вправо на один импульс  
3. **AUTO / MANUAL** - переключение между автоматическим и ручным режимами

## 🚀 Подключение и использование

### 1️⃣ Включите устройство
После включения ESP32-C3 начнется инициализация WebServer:
```
=== WiFi Mode (Access Point) ===
AP SSID: SolarTracker
AP Password: 12345678
AP IP Address: 192.168.4.1
```

### 2️⃣ Подключитесь к WiFi
- На вашем устройстве (телефон, ноутбук, планшет)
- Откройте список доступных WiFi сетей
- Найдите сеть **"SolarTracker"**
- Подключитесь с паролем **"12345678"**

### 3️⃣ Откройте веб-интерфейс
- Откройте браузер
- Перейдите на адрес: **http://192.168.4.1**
- Вы увидите красивый интерфейс управления

### 4️⃣ Используйте управление
- Нажимайте кнопки LEFT/RIGHT для ручного управления
- Переключайтесь между режимами AUTO/MANUAL
- Смотрите показания напряжений в реальном времени

## 🔌 REST API

Веб-интерфейс использует REST API для обмена данными:

### GET /api/status
Получить текущие показания в формате JSON

**Ответ:**
```json
{
  "vLeft": 345.2,      // Напряжение левой панели (мВ)
  "vRight": 342.1,     // Напряжение правой панели (мВ)
  "mode": 0,           // 0=AUTO, 1=MANUAL, 2=CALIBRATION
  "motorDir": 0        // 0=STOP, 1=LEFT, 2=RIGHT
}
```

### POST /api/cmd/left
Команда поворота влево

### POST /api/cmd/right
Команда поворота вправо

### POST /api/cmd/mode
Команда переключения режима AUTO/MANUAL

## 🛠️ Конфигурация

Для изменения параметров WiFi отредактируйте файл `include/webserver.h`:

```cpp
#define WIFI_SSID     "SolarTracker"    // Имя точки доступа
#define WIFI_PASSWORD "12345678"        // Пароль (минимум 8 символов)
#define WIFI_CHANNEL  1                 // Канал WiFi
```

## 🐛 Решение проблем

### Не могу подключиться к WiFi
- Убедитесь, что устройство включено
- Проверьте указанный SSID и пароль
- Перезагрузитесь
- Убедитесь, что рядом нет помех на частоте 2.4 ГГц

### Интерфейс не загружается
- Проверьте адрес: http://192.168.4.1
- Попробуйте 192.168.4.1:80
- Очистите кеш браузера (Ctrl+F5)
- Используйте другой браузер

### Кнопки не работают
- Проверьте подключение к WiFi
- Откройте консоль браузера (F12) для ошибок
- Проверьте питание устройства

## 📝 Примеры использования

### JavaScript (получение данных)
```javascript
fetch('http://192.168.4.1/api/status')
  .then(r => r.json())
  .then(data => console.log(`Left: ${data.vLeft}mV, Right: ${data.vRight}mV`));
```

### Python (управление)
```python
import requests

# Поворот влево
requests.post('http://192.168.4.1/api/cmd/left')

# Переключение режима
requests.post('http://192.168.4.1/api/cmd/mode')

# Получение данных
data = requests.get('http://192.168.4.1/api/status').json()
print(f"Left voltage: {data['vLeft']} mV")
```
