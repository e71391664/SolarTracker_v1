# История развития и изменений (Changelog)

Все значимые изменения проекта SolarTracker будут задокументированы в этом файле.

## Формат

Версионирование следует принципам [Semantic Versioning](https://semver.org/).

- **[MAJOR]** - несовместимые изменения API
- **[MINOR]** - новые функции, совместимые с предыдущими версиями
- **[PATCH]** - исправления ошибок
- **[DOCS]** - обновления документации
- **[CHORE]** - обновления конфигурации и зависимостей
