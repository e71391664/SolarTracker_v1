# SolarTracker Project ☀️

## Описание
SolarTracker — это проект на основе PlatformIO для платы **Seeed XIAO ESP32C3**, разработанный для отслеживания положения солнца и управления солнечными панелями. Поддерживает управление через **встроенный веб-интерфейс via WiFi**.

## 🚀 Быстрый старт

### Подключение через WiFi
1. **Включите устройство** - оно создаст точку доступа WiFi
2. **Подключитесь**:
   - SSID: `SolarTracker`
   - Пароль: `12345678`
   - IP: `192.168.4.1`
3. **Откройте браузер** - перейдите на `http://192.168.4.1`
4. **Управляйте** - нажимайте кнопки LEFT/RIGHT/MODE для управления

### Кнопки управления
- ⬅️ **LEFT** - поворот влево
- ➡️ **RIGHT** - поворот вправо  
- 🔄 **AUTO/MANUAL** - переключение режимов

## 📊 Возможности

### Автоматическое позиционирование
- Дифференциальный трекинг по разнице напряжений
- Два режима: AUTO (автоматический) и MANUAL (ручной)
- Режим калибровки для выравнивания панелей

### Мониторинг
- Real-time показания обеих панелей (мВ)
- Визуализация разницы напряжений
- Отслеживание статуса мотора

### WiFi управление (v1.1.0+)
- 📱 Веб-интерфейс в браузере
- 🌐 REST API для программирования
- ⚡ Real-time обновление данных (500 мс)
- 📱 Адаптивный дизайн для мобильных устройств

## 🛠️ Техническая спецификация

### Оборудование
- **Плата**: Seeed XIAO ESP32C3 (встроенный WiFi)
- **Датчики**: 2x INA226 (напряжение панелей)
- **Дисплей**: OLED 128x64 SSD1306
- **Мотор**: L298N DC драйвер
- **Питание**: 5V USB
- **Интерфейсы**: I2C, GPIO, PWM

### Характеристики трекера
- **Напряжение панелей**: 0-17 мВ
- **Пороговое напряжение**: 150 мВ
- **Скорость мотора**: 180 PWM (256 макс)
- **Время импульса**: 600 мс
- **Интервал замеров**: 500 мс

## 💻 REST API

### Получить данные
```bash
curl http://192.168.4.1/api/status
```

**Ответ**:
```json
{
  "vLeft": 345.2,      // мВ левой панели
  "vRight": 342.1,     // мВ правой панели
  "mode": 0,           // 0=AUTO, 1=MANUAL, 2=CALIBRATION
  "motorDir": 0        // 0=STOP, 1=LEFT, 2=RIGHT
}
```

### Управление
```bash
curl -X POST http://192.168.4.1/api/cmd/left   # Влево
curl -X POST http://192.168.4.1/api/cmd/right  # Вправо
curl -X POST http://192.168.4.1/api/cmd/mode   # Переключить режим
```

## 🔌 Пины ESP32-C3

```
I2C:
  SDA → GPIO 8
  SCL → GPIO 9

L298N (мотор):
  IN1 → GPIO 4
  IN2 → GPIO 5
  ENA → GPIO 6

Кнопки:
  AUTO/MANUAL  → GPIO 7
  LEFT         → GPIO 2
  RIGHT        → GPIO 3
```

## 🐛 Решение проблем

### Не видна WiFi сеть
- Проверьте питание USB
- Перезагрузитесь
- Проверьте частоту 2.4 ГГц

### Не загружается веб-интерфейс
- Попробуйте `http://192.168.4.1:80`
- Очистите кеш браузера (Ctrl+F5)
- Проверьте подключение в серийном мониторе

### Команды не работают
- Проверьте подключение к WiFi
- Посмотрите консоль браузера (F12)
- Проверьте питание устройства
