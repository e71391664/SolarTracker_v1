# Prompt: Stepper Driver Sketch for Raspberry Pi Pico and Arduino Nano

## Цель
Напиши скетч на языке C для управления шаговым двигателем c редуктором (производитель Kokusan Denki) через драйвер SLA6023 + контролер частоты KA20885D  с кварцем.
Код должен работать в среде PlatformIO и быть легко адаптируемым для двух плат:
- Raspberry Pi Pico
- Arduino Nano

## Описание устройства
Драйвер SLA6023 принимает два основных сигнала:
- `SLK` / `PULSE` — шаговый сигнал
- `DIR` — направление вращения

Управление выполняется двумя кнопками:
- `LEFT_BUTTON_PIN` — поворот влево
- `RIGHT_BUTTON_PIN` — поворот вправо

При отпускании кнопки двигатель должен останавливаться мгновенно.

## Общие константы и параметры
В коде должны быть вынесены следующие константы:
- `STEP_PIN` — вывод шага (`SLK` / `PULSE`)
- `DIR_PIN` — вывод направления (`DIR`)
- `LEFT_BUTTON_PIN` — кнопка «влево»
- `RIGHT_BUTTON_PIN` — кнопка «вправо»
- `STEPS_180_DEGREES` — число импульсов для поворота на 180°
- `STEP_DELAY_US` — задержка в микросекундах между полуволнами шага
- `BUTTON_DEBOUNCE_MS` — задержка для подавления дребезга кнопок

## Пины для Arduino Nano
- `STEP_PIN = D3`
- `DIR_PIN = D4`
- `LEFT_BUTTON_PIN = D8`
- `RIGHT_BUTTON_PIN = D9`

## Пины для Raspberry Pi Pico
Для Pico используйте свободные GPIO, но оставьте в коде общие константы.
Например:
- `STEP_PIN = 3`  (GPIO 3)
- `DIR_PIN = 4`   (GPIO 4)
- `LEFT_BUTTON_PIN = 8`  (GPIO 8)
- `RIGHT_BUTTON_PIN = 9` (GPIO 9)

## Алгоритм
1. Инициализировать пины: шаг и направление как выходы, кнопки как входы с подтяжкой.
2. Проверять состояние кнопок в цикле.
3. Если нажата кнопка «влево»:
   - установить направление вращения для левого хода
   - генерировать шаги до достижения `STEPS_180_DEGREES`
4. Если нажата кнопка «вправо»:
   - установить направление для правого хода
   - генерировать шаги до достижения `STEPS_180_DEGREES`
5. Если нажата ни одна кнопка, не подавать шаговые импульсы.
6. При отпускании кнопки генерация шагов должна прекращаться немедленно.

## Особенности реализации
- Двигатель шаговый, с внутренним редуктором и неизвестным коэффициентом редукции.
- `STEPS_180_DEGREES` должен быть легко настраиваемой константой.
- Шаговый сигнал формируется через `digitalWrite(HIGH)`, `delayMicroseconds`, `digitalWrite(LOW)`, `delayMicroseconds`.
- Поведение должно быть простым и понятным.

- алгоритм управления
